# Распознавание пропусков СПБГУ

## Описание проекта

Данное приложение предназначено для:

1. Определения, является ли изображение пропуском СПБГУ (с помощью нейросети).
2. Распознавания номера пропуска (с использованием OCR).

Проект состоит из:

* модели компьютерного зрения на базе InceptionV3
* десктопного приложения на Tkinter
* OCR-распознавания через EasyOCR

---

## Используемые технологии

* Python 3.x
* TensorFlow / Keras
* OpenCV
* EasyOCR
* Tkinter
* Pillow

## Обучение модели

Файл: `train.py`

Модель основана на transfer learning (InceptionV3):

* Заморожены первые 205 слоев
* Добавлены полносвязные слои
* Используется функция потерь `binary_crossentropy`

Запуск:

```bash
python train.py
```

После обучения модель сохранится в файл:

```
CardsDetectorNew.keras
```

---

## Функционал приложения:

* выбор изображения
* отображение изображения
* определение: пропуск / не пропуск
* распознавание номера

---

## Как работает модель

1. Изображение приводится к размеру 290x290
2. Передаётся в InceptionV3
3. Модель возвращает вероятность:

   - Если > 0.5  → пропуск

   - ≤ 0.5 → не пропуск

---

## OCR (распознавание номера)

Используется EasyOCR:

* язык: русский (`ru`)
* извлекаются цифры длиной 4–8 символов
